## 🔠 Getting Started  

### Prerequisites  
- Node.js and npm installed  

### Installation  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/your-username/portfolio.git  
   ```  
2. Navigate to the project directory:  
   ```bash  
   cd portfolio  
   ```  
3. Install dependencies:  
   ```bash  
   npm install  
   ```  

### Development  
Start the development server:  
```bash  
npm start  
```
